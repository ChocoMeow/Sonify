# Installation

Make Environment (Use Python 3.9 - 3.11 version)
```
# Generate Environment
python3 -m venv MusicGen

# Activate The Environment That You Just Created
source ./MusicGen/bin/activate # Only for MacOs / Linux

```

Install the required module (Torch and Python Must be Installed First)
```
pip install torch==2.0.0 torchvision==0.15.1 torchaudio==2.0.1 # CPU Version
pip install -r requirements.txt

```
